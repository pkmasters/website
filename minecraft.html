How to Join Our Minecraft Server

Follow the steps below to join our server on Java or Bedrock Edition.

For Java Edition Players

  1. Launch Minecraft Java Edition (version 1.20 or higher).
  2. Click on Multiplayer and then Add Server.
  3. Enter the following details:
    • Server Name: Your Server Name
    • Server Address: play.xoiph.com
  4. Click Done and then select the server to join.

For Bedrock Edition Players

  1. Launch Minecraft Bedrock Edition.
  2. Go to Play and click Servers tab.
  3. Scroll down and click Add Server.
  4. Enter the following details:
    • Server Name: Your Server Name
    • Server Address: play.xoiph.com
    • Port: 1546
  5. Click Save and then select the server to join.
